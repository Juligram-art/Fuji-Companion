FUJI PASO A PASO — versión inicial

Contenido:
- Interfaz nocturna y de alto contraste.
- Botones grandes para usar con poca luz.
- Seis situaciones fotográficas.
- Preguntas breves para adaptar los ajustes.
- Una sola instrucción visible en cada paso.
- Funcionamiento sin conexión después de la primera carga.

Publicación en GitHub Pages:
1. Crea un repositorio nuevo o abre uno existente.
2. Pulsa Add file > Upload files.
3. Sube todos los archivos de esta carpeta, sin subir la carpeta contenedora.
4. Abre Settings > Pages.
5. En Source selecciona Deploy from a branch.
6. Selecciona main y /(root), y guarda.
7. Abre la URL publicada desde Safari en el iPhone.
8. Pulsa Compartir > Añadir a pantalla de inicio.

Nota: esta guía es un prototipo práctico. Los nombres exactos de algunos menús pueden variar con el firmware o la personalización de botones de la cámara.
